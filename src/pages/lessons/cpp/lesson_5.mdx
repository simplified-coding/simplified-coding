# Μάθημα 5ο: Δηλώσεις εναλλαγής (switch)

## Δήλωση εναλλαγής (switch)

Μια δήλωση εναλλαγής περιέχει μια λίστα από περιπτώσεις (cases)
και έναν ή περισσότερους εκφραστές (expressions) που πρέπει να αξιολογηθούν.
Κάθε εκφραστής αντιστοιχεί με μια περίπτωση, και εκτελεί τις εντολές που συνοδεύουν την περίπτωση.

Ας δούμε ένα παράδειγμα για να καταλάβουμε καλύτερα:

### Παράδειγμα

```c++
#include <iostream>

using namespace std;

int main() {
    int day = 5;

    switch (day) {
        case 1:
            cout << "Δευτέρα";
            break;
        case 2:
            cout << "Τρίτη";
            break;
        case 3:
            cout << "Τετάρτη";
            break;
        case 4:
            cout << "Πέμπτη";
            break;
        case 5:
            cout << "Παρασκεύη";
            break;
        case 6:
            cout << "Σάββατο";
            break;
        case 7:
            cout << "Κυριακή";
            break;
        default:
            cout << "Μη έγκυρη τιμή";
    }

    return 0;
}
```

Ας αναλύσουμε βήμα-βήμα το παραπάνω πρόγραμμα:

#### `switch (day)`

Η δήλωση `switch` ξεκινά με τη μεταβλητή `day` ως εκφραστή.
Αυτό σημαίνει ότι ο εκφραστής θα αξιολογηθεί και θα συγκριθεί με κάθε περίπτωση.
Θα εκτελεστεί έτσι ο κώδικας της περίπτωσης που ταιριάζει με την τιμή της μεταβλητής.

#### `case 1:` μέχρι `case 7:`

Οι περιπτώσεις είναι οι τιμές που μπορεί να πάρει η μεταβλητή `day`.
Αν η τιμή της μεταβλητής `day` είναι ίση με την τιμή της περίπτωσης, τότε ο κώδικας της περίπτωσης εκτελείται.

Εδώ λοιπόν, αν η τιμή της μεταβλητής `day` είναι `1`, τότε θα εκτελεστεί ο κώδικας της περίπτωσης `case 1:`
και θα εμφανιστεί στην οθόνη το μήνυμα `Δευτέρα`.

#### `break`

Η εντολή `break;` χρησιμοποιείται για να σταματήσει η εκτέλεση των εντολών της περίπτωσης.
Αν δε χρησιμοποιηθεί η εντολή `break;`, τότε θα εκτελεστούν και οι εντολές της επόμενης περίπτωσης.

Η εντολή αυτή θα μας είναι ιδιαίτερα χρήσιμη σε επόμενα μαθήματα.

#### `default:`

Η περίπτωση `default:` χρησιμοποιείται όταν η τιμή της μεταβλητής `day` δεν ταιριάζει με καμία από τις περιπτώσεις.
