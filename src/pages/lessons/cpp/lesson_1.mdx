<h2 style="text-align: center;"> Βασική σύνταξη </h2>

Κάθε πρόγραμμα στη C++ αρχίζει έτσι:

```cpp
  #include <iostream>
  using namespace std;

  int main() {

    return 0;
  }
```

Ας δούμε τι κάνει η κάθε γραμμή:

<h3 style="text-align: left;"> 
```cpp 
#include <iostream>
``` 
</h3>

Η γραμμή αυτή λέει στη C++ να προσθέσει τη βιβλιοθήκη `iostream` στο πρόγραμμα.

Η βιβλιοθήκες είναι συλλογές από κώδικα που έχει γραφτεί από άλλους προγραμματιστές
και περιέχει κώδικα που μπορούμε να χρησιμοποιήσουμε στο πρόγραμμά μας.
Η βιβλιοθήκη αυτή περιέχει τις βασικές λειτουργίες εισόδου/εξόδου της C++ που θα δούμε σε λίγο.

<h3 style="text-align: left;"> 
```cpp 
using namespace std;
``` 
</h3>

Η γραμμή αυτή λέει στη C++ να χρησιμοποιήσει το namespace `std`. Το namespace `std`
περιέχει τις βασικές λειτουργίες της C++ και είναι η βασική βιβλιοθήκη που χρησιμοποιούμε
στα προγράμματά μας. Το namespace `std` περιέχει πολλές λειτουργίες και αν δε χρησιμοποιήσουμε
την παραπάνω γραμμή, τότε θα πρέπει να γράψουμε το όνομα του namespace πριν από κάθε λειτουργία
που θέλουμε να χρησιμοποιήσουμε. Έτσι, γράφουμε `using namespace std;` για να μη χρειάζεται
να γράφουμε `std::cout` σε βασικές εντολές. Βέβαια, σε προχωρημένα προγράμματα, αυτό μπορεί να
κάνει το πρόγραμμά μας πιο αργό. Εμείς όμως δε χρειάζεστε να ανησυχούμε για αυτό προς το παρόν.

<h3 style="text-align: left;"> 
```cpp 
int main() { ... }
``` 
</h3>

Η γραμμή αυτή είναι η συνάρτηση `main`. Όλα τα προγράμματα στη C++ ξεκινάνε από αυτή τη συνάρτηση.
Θα δούμε τι είναι οι συναρτήσεις σε άλλο μάθημα. Αυτό που πρέπει να ξέρουμε τώρα είναι ότι όλα τα προγράμματα
στη C++ ξεκινάνε από αυτή τη συνάρτηση και ότι η συνάρτηση αυτή πρέπει να επιστρέφει έναν ακέραιο αριθμό.

<h3 style="text-align: left;"> 
```cpp 
return 0;
``` 
</h3>

Η γραμμή αυτή είναι η εντολή `return`. Η εντολή αυτή επιστρέφει έναν ακέραιο αριθμό από τη συνάρτηση `main`.
Αυτός ο ακέραιος αριθμός είναι ο κώδικας εξόδου του προγράμματος. Ο κώδικας εξόδου `0` σημαίνει ότι το πρόγραμμα
τελείωσε κανονικά. Αν ο κώδικας εξόδου είναι διαφορετικός του `0`, τότε σημαίνει ότι το πρόγραμμα τερματίστηκε
με κάποιο σφάλμα.

<h3 style="text-align: left;"> Προσοχή </h3>

Κάθε εντολή στη C++ απαιτεί ένα `;` στο τέλος της.
Αν ξεχάσουμε να βάλουμε το `;` στο τέλος μιας εντολής,
τότε το πρόγραμμά μας δε θα μπορεί να τρέξει.
Το `;` δεν είναι Ελληνικό ερωτηματικό αλλά το αντίστοιχο της
άνω τελείας στα Αγγλικά.
Αυτό σημαίνει ότι αν το γράψουμε με Ελληνικό πληκτρολόγιο η C++ δε θα το αναγνωρίσει ώς
άνω τελεία και θα μας βγάλει σφάλμα.
Γι'αυτό πρέπει να το γράφουμε πάντα με Αγγλικό πληκτρολόγιο.

<h2 style="text-align: center;"> Σχόλια </h2>

Στη C++ μπορούμε να γράψουμε σχόλια στον κώδικά μας. Τα σχόλια είναι κομμάτια κώδικα που δεν εκτελούνται
από τον υπολογιστή. Τα σχόλια χρησιμοποιούνται για να εξηγήσουμε τον κώδικά μας σε άλλους προγραμματιστές
ή σε εμάς τους ίδιους αν τον ξαναδούμε αργότερα. Αυτό είναι πολύ χρήσιμο όταν ο κώδικάς μας είναι πολύ 
εκτενής ή/και πολύπλοκος.

Υπάρχουν δύο είδη σχολίων στη C++ (ακριβώς όπως έχουμε δει και στην Python):

* Μονογραμμικά:

  Τα μονογραμμικά σχόλια είναι σχόλια που αρχίζουν με `//` και τελειώνουν στο τέλος της γραμμής.
  ```cpp
  // Αυτό είναι ένα μονογραμμικό σχόλιο
  ```

* Πολυγραμμικά

  Τα πολυγραμμικά σχόλια είναι σχόλια που αρχίζουν με `/*` και τελειώνουν με `*/`.
  ```cpp
  /* Αυτό είναι ένα πολυγραμμικό σχόλιο
    που μπορεί να εκτείνεται σε πολλές γραμμές */
  ```

<h2 style="text-align: center;"> Εκτύπωση στην οθόνη (εντολή `cout`) </h2>

Η εντολή `cout` είναι η εντολή που χρησιμοποιούμε για να εκτυπώσουμε κάτι στην οθόνη (κονσόλα) του χρήστη.

Για να εκτυπώσουμε κάτι στην οθόνη, γράφουμε:
```cpp
cout << "Hello World";
```

Ας δούμε ένα παράδειγμα:
```cpp
#include <iostream>
using namespace std;

int main() {
  cout << "Hello World!!" << endl;
  return 0;
}
```

Το παραπάνω πρόγραμμα θα εκτυπώσει στην οθόνη το κείμενο `Hello World!!`.
Όταν θέλουμε να εκτυπώσουμε ένα κείμενο, τότε το κείμενο αυτό πρέπει να βάλουμε μέσα σε εισαγωγικά `"`.
Θα δούμε περισσότερα για την εκτύπωση στην οθόνη και για το τι μπορείς να εκτυπώσεις σε επόμενο μάθημα (3ο).
