<h2 style="text-align: center;">If</h2>

```
if <προϋπόθεση>:
    <αποτέλεσμα>
```

Εάν η προϋπόθεση ισχύει (όπως στην πρώτη περίπτωση), τότε το αποτέλεσμα θα πραγματοποιηθεί. 

Εάν η προϋπόθεση δεν ισχύει (όπως στην δεύτερη περίπτωση), τότε το αποτέλεσμα δεν θα πραγματοποιηθεί. 

```python
example = True
if example == True:
    print("hello")
hello

example = False
if example == True:
    print("hello")


```

<h2 style="text-align: center;">Elif</h2>

Εάν μία συνθήκη/προϋποθεση δεν ισχύει, μπορούμε να ελέξουμε αν ισχύει μία άλλη συνθήκη.
Αυτή την δυνατότητα μας την προσφέρει το ```elif```.


```python
example = False
if example == True:
    print("hello")

elif example == False:
    print("hello_2")

hello_2

```
**Προσοχή!!!** Το elif πρέπει να μπαίνει ακριβός κάτω από ένα ```if``` ή από ένα ```elif```. 

---

<h2 style="text-align: center;"> Else </h2>

Στην περίπτωση που καμία από τις προηγούμενες προϋποθέσεις δεν ισχύει, τότε ένα ```else``` ,που θα έχουμε βάλει, θα πραγματοποιηθεί.

```python
example = 15

if example <= 10:
    print("Hello")
elif example == 10:
    print("Hello_2")
else:
    print("Hello_3")

Hello_3
```
